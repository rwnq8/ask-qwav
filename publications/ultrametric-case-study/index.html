Abstract

We present a production implementation of ultrametric distance and p-adic valuation principles applied to a real-world information retrieval system: the QWAV research paper corpus "Did You Mean?" discovery engine. The system uses agglomerative single-linkage clustering — the only hierarchical clustering method that guarantees ultrametric distances — to organize 451 academic paper titles into a binary dendrogram. A three-phase discovery engine (word-level edit matching → ultrametric cluster expansion → tree-based pruned search) enables cross-disciplinary paper discovery beyond direct word overlap. The architecture demonstrates five p-adic and ultrametric principles in production, with the tree persisted to Cloudflare R2 for cold-start resilience and visualized via an interactive Pages dendrogram.

1. Mathematical Foundation

1.1 Ultrametric Distance

An ultrametric distance d satisfies the strong triangle inequality:

d(x,z) ≤ max(d(x,y), d(y,z))    ∀x,y,z ∈ X

This is strictly stronger than the standard metric inequality. All triangles are isosceles, balls are either disjoint or nested, and ultrametric spaces correspond canonically to rooted trees (dendrograms) where the distance between two leaves equals the height of their lowest common ancestor.

1.2 p-adic Valuation

The p-adic valuation ord_p(x) is the largest k such that p^k divides x. The p-adic absolute value is |x|_p = p^{-ord_p(x)}. In our system, we approximate ord_p(paper) as the depth of the paper leaf from the dendrogram root — papers with low depth are more "foundational," while papers with high depth are more "specialized."

1.3 Ostrowski's Theorem

Every non-trivial absolute value on ℚ is equivalent to either the usual absolute value |·|_∞ or a p-adic absolute value |·|_p.

We combine both into an Ostrowski hybrid score — a weighted harmonic mean of the Euclidean-like cluster-distance norm and the 2-adic valuation norm. This gives a mathematically principled ranking that balances structural position with hierarchical depth.

1.4 Hensel's Lemma

If f(x) ≡ 0 (mod p) and f'(x) ≢ 0 (mod p), then there exists a unique lift to a root modulo pⁿ for all n.

Search results are organized into Hensel layers by ultrametric merge distance. Each layer represents one "lift" deeper into the knowledge hierarchy, enabling incremental search refinement.

2. Implementation Architecture

2.1 Tree Construction

Algorithm: buildUltrametricTree(titles)
  Input:  n = 451 paper titles
  Output: Root node of ultrametric dendrogram

  1. Create n leaf nodes
  2. Compute n×n Levenshtein distance matrix (Float64)
  3. While active clusters > 1:
     a. Find pair (i,j) with minimum D[i][j]
     b. Merge into internal node with distance = D[i][j]
     c. Update D via single-linkage: D[new][k] = min(D[i][k], D[j][k])
  4. Return root

  Complexity: O(n³) ≈ 15M ops, executes in ~2-3s on Cloudflare Workers

Phase 2 is the key innovation — papers that are mathematically "near" in the ultrametric tree are suggested even when they lack direct word overlap with the query.

2.3 Search Pruning via Strong Triangle Inequality

searchNode(node, query, maxDistance):
  for each child of internal node:
    d_rep = levenshtein(query, child.rep)
    effectiveRadius = child.distance
    if d_rep ≤ maxDistance + effectiveRadius:
      searchNode(child)  // subtree may contain matches
    else:
      PRUNE  // by strong triangle inequality, no member can match
